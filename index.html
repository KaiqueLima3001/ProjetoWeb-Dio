HTML: A Linguagem da Estruturação de Conteúdo na Web

Tag <h1> - Título Principal

A tag <h1> é usada para definir o título principal de uma página. Geralmente, há apenas um <h1> por página, pois representa o tópico principal do conteúdo.

Tag <h2> - Título de Seção

A tag <h2> é usada para criar títulos de seção dentro de uma página. Ela é usada para organizar o conteúdo em partes menores e hierarquicamente.

Tag <h3> - Título de Subseção

A tag <h3> é semelhante ao <h2>, mas é usada para criar títulos de subseções dentro das seções definidas com <h2>.

Aqui estão outras tags importantes em HTML:

  • <p> - Parágrafo: A tag <p> é usada para criar parágrafos de texto. Ela é essencial para formatar o texto em blocos.(Leia mais sobre a tag <p> aqui)
  • <mark> - Destaque: A tag <mark> é usada para destacar parte do texto, como um termo importante.
  • <small> - Texto Pequeno: A tag <small> é usada para diminuir o tamanho do texto, tornando-o menos proeminente.
  • <i> - Texto Em Itálico: A tag <i> é usada para enfatizar o texto, geralmente exibindo-o em itálico.
  • <u> - Texto Sublinhado: A tag <u> é usada para sublinhar o texto, indicando ênfase ou links.
  • <font> - Fonte: A tag <font> foi usada no passado para definir estilos de fonte diretamente no HTML. No entanto, seu uso é desencorajado em favor do CSS para estilos de fonte mais modernos.(Leia mais sobre a tag <font> aqui)
  • <del> - Texto Excluído: A tag <del> é usada para representar texto que foi deletado ou riscado. Ela é frequentemente usada em documentos revisados ou alterados.(Leia mais sobre a tag <del> aqui)
  • <abbr> - Abreviação: A tag <abbr> é usada para definir abreviações e acrônimos, permitindo que os navegadores forneçam informações adicionais quando o usuário passa o mouse sobre o texto abreviado.(Leia mais sobre a tag <abbr> aqui)
